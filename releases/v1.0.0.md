This is the first official release of the AI Workshop Teacher repository, providing a complete Kubernetes-based environment for running AI workshops with TIBCO ActiveSpaces, Ollama, and document processing capabilities.

## ‚ú® Features

### Complete Workshop Stack
- **ActiveSpaces (AS-VDB-gRPC)**: Document store and vector database with gRPC services
- **Ollama**: LLM inference (Llama 3.2 3B) and embeddings (Nomic-Embed-Text) for RAG applications
- **Docling**: Document ingestion and processing service
- **OpenWebUI**: Web-based interface for LLM interaction and testing

### Helm Charts
All components are deployable via Helm charts with production-ready configurations:
- `helm/as-vdb-grpc` - TIBCO ActiveSpaces vector database services
- `helm/ollama` - Ollama LLM and embedding server with CPU-optimized settings
- `helm/docling` - Docling document processing service
- `helm/openwebui` - OpenWebUI interface with pipelines and Redis support

## üöÄ Quick Deploy

```bash
# Deploy ActiveSpaces
cd helm/as-vdb-grpc
helm install activespaces . -f values-aks.yaml --namespace ai --create-namespace

# Deploy Ollama
cd helm/ollama
helm install ollama . -f values-aks.yaml --namespace ai --create-namespace
kubectl scale deployment -n ai ai-workshop-ollama --replicas=1

# Deploy Docling
cd helm/docling
helm install docling . -f values.yaml --namespace ai --create-namespace

# Deploy OpenWebUI
cd helm/openwebui
helm install openwebui . -f values-aks.yaml --namespace ai --create-namespace
```
